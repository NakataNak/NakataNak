# Yuki Nakata / 中田 裕貴
京都大学 数学教室（数学・数理解析専攻 数学系） の[博士課程学生](https://www.math.kyoto-u.ac.jp/ja/member/graduate)です。
## 研究主題
整数論の中で，簡約群の被覆群の表現論に取り組んでいます。
